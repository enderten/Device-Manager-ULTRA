# 💎 Device Manager Pro (Liquid Glass Edition)
A high-performance Android device manager (ADB/Fastboot) with a stunning Liquid Glass interface for macOS.

## ✨ Features
* **Liquid Glass UI:** Fully native macOS frosted glass panels with dynamic fading effects.
* **ADB Manager:** Execute commands, check devices, and reboot to bootloader instantly.
* **Fastboot Pro:** Flash partitions (`system`, `vendor`, `boot`, `vbmeta`) with smart filters.
* **VBMeta Guard:** Toggle `--disable-verity` and `--disable-verification` with one click.
* **Safe Sideload:** Extension-locked file picker (only `.zip`) to prevent terminal errors.

## :) Installation
1. Go to the [Releases](https://github.com/enderten/Device-Manager-Pro/releases) section.
2. Download the `.zip` archive.
3. Extract and move the `.app` to your **Applications** folder.

## 🛠 Requirements
* **OS:** macOS 14.0 (Sonoma) or newer.
* **Tools:** `adb` and `fastboot` must be installed in your system path (Homebrew recommended).
